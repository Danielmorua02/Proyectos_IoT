Este proyecto consiste en configurar los puertos de entrada y salida de una Raspberry Pi 4 model B.
Para implementar el sistema de proximidad es necesario que cuentes con la libreria gpiozero (por defecto los sistemas operativos Raspberry OS actuales cuentan con ella).

Si tu dispositivo no tiene instalada esta libreria basta con que ejecutes los siguientes pasos:
	
	- Instala "python3-gpiozero" usando "apt-get"
	- Actualiza la base de datos apt con "apt-get" usando el siguiente comando.
	- "sudo apt-get update" (Ejecutarlo sin las comillas)
	- Despues de actualizar la base de datos apt, podremos instalar "python3-gpiozero" usando "apt-get" ejecutando el siguiente comando:
	- "sudo apt-get -y install python3-gpiozero" (Ejecutarlo sin las comillas)
